===============================
📘 시스템 메뉴얼 (v1.0)
===============================

1. 🛠 설치 가이드
----------------
- 필요한 소프트웨어: Python 3.10+, Git, Docker
- 설치 단계:
  1) GitHub에서 소스코드 클론: `git clone <repo_url>`
  2) 가상환경 설정: `python -m venv venv`
  3) 의존성 설치: `pip install -r requirements.txt`

2. 🚀 실행 방법
----------------
- 서버 실행: `python main.py`
- 웹 접속: 브라우저에서 `http://localhost:8000`

3. ⚙ 주요 기능
----------------
- 데이터 수집: API 연동으로 데이터 크롤링 가능
- 분석 기능: 시각화 및 통계 기능 지원
- 배포: Docker 이미지 빌드 및 컨테이너 실행

4. 🔧 문제 해결
----------------
- 포트 충돌 시:
  - 다른 포트 사용: `python main.py --port 8080`
- 모듈 오류:
  - `pip install --upgrade <module_name>`

5. 📞 문의
----------------
- 관리자 이메일: admin@example.com
- 버그 리포트: GitHub Issues 활용
